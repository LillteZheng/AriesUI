# AriesUI
一个常用的UI工具类，用于快速定制

```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```
```
implementation 'com.github.LillteZheng:AriesUI:v1.07'
```
